# PaprFeed

PaprFeed is a calm, mobile-friendly research discovery app for browsing recent scientific papers without the noise of a traditional social feed.

Visit the app: [paprfeed.com](https://paprfeed.com)

## Research Sources

- arXiv
- bioRxiv
- medRxiv
- PubMed

## Status

PaprFeed is under active development.
